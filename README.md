 Energy Disaggregation Platform:

A full-stack system for breaking down whole-home energy usage into appliance-level estimates using event detection, segmentation, feature extraction, and machine-learning classification.
This project consists of:
Frontend (GitHub Pages)
Interactive dashboard (HTML/JS/CSS) displaying:
Appliance breakdown pie chart
Confidence levels
24-hour usage chart
Daily/weekly/monthly estimates
CSV upload + automatic processing
